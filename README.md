# Learn Playwright — Batch 2x

A hands-on JavaScript fundamentals course building up to Playwright automation. All examples are standalone Node.js scripts — no dependencies required.

## Contents

### Chapter 1: Basics
| File | Topic |
|------|-------|
| `01_Basics.js` | `console.log()` — Hello World |
| `02_JS.js` | `let`, `for` loops, function declarations, hoisting |
| `03_JS_Verify_Setup.js` | Node.js env check — `process.platform`, `process.arch`, `process.version` |
| `04_HotCode.js` | Functions with parameters/return, loop iteration, JIT / hot code concept |

### Chapter 2: JavaScript Concepts
| File | Topic |
|------|-------|
| `05_JS_Basics.js` | `var` keyword and variable reassignment |

### Chapter 3: Identifiers, Literals & Naming Conventions
| File | Topic |
|------|-------|
| `06_Identifier_Rules.js` | Valid identifier characters (`$`, `_`, letters, digits), case sensitivity |
| `07_Identifier_Part2.js` | Naming conventions — camelCase, PascalCase, snake_case, SCREAMING_SNAKE_CASE, Hungarian Notation |
| `08_Comments.js` | Single-line, multi-line, and JSDoc comments |
| `js_identifier_rules.js` | Comprehensive identifier rules with Unicode support and console output |
| `all_identifier_examples.js` | Consolidated identifier examples from `06` and `07` |
| `VS_Code_keyboard_shortcut_mac.md` | VS Code keyboard shortcuts for macOS |
| `VS_Code_keyboard_shortcut_windows.md` | VS Code keyboard shortcuts for Windows |

## Running Examples

```bash
node "chapter_01_Basics/01_Basics.js"
node "chapter_02_Javascript_Concepts/05_JS_Basics.js"
node "chapter_03_Identifier_Literals/06_Identifier_Rules.js"
```
